Determinantes

A cada matriz cuadrada se le puede asociar un único número: su determinante. Ese número dice si la matriz es invertible, cuánta "área" o "volumen" genera, y si un sistema de ecuaciones tiene solución única.

¿Sabías qué? En ingeniería civil, un determinante distinto de cero en el sistema de fuerzas de una estructura indica solución única — la estructura es estable. En electrónica, lo mismo pasa al resolver corrientes y voltajes de un circuito con varias mallas.

1. Determinante 2×2

Para una matriz A de 2×2, el determinante ya lo conocías: es un producto cruzado con resta.

det(A) para 2×2 A = (a₁₁ a₁₂ ; a₂₁ a₂₂)  ⇒  det A = |A| = a₁₁a₂₂ − a₂₁a₁₂

Un resultado clave que ya viste con matrices inversas: A es invertible si y sólo si det A ≠ 0. Los determinantes de matrices más grandes se definen apoyándose en este caso base, de forma inductiva (se usa 2×2 para definir 3×3, 3×3 para definir 4×4, y así siguiendo).

2. Determinante 3×3 — expansión por cofactores

Para una matriz de 3×3, el determinante se calcula "abriendo" por la primera fila, multiplicando cada elemento por el determinante 2×2 que queda al tachar su fila y columna (alternando signos +, −, +):

det(A) para 3×3 det A = a₁₁|a₂₂ a₂₃; a₃₂ a₃₃| − a₁₂|a₂₁ a₂₃; a₃₁ a₃₃| + a₁₃|a₂₁ a₂₂; a₃₁ a₃₂|
✏️ Ejemplo

A = (4 7 2; 3 −5 1; −8 6 9)

det A = 4[(−5)(9)−(6)(1)] − 7[(3)(9)−(−8)(1)] + 2[(3)(6)−(−8)(−5)]

det A = 4(−51) − 7(35) + 2(−22) = −204 − 245 − 44 = −493

Atajo para 3×3 — Regla de Sarrus

Sólo para matrices 3×3 existe un atajo: se repiten las dos primeras columnas a la derecha, se multiplican las tres diagonales que bajan (con signo +) y las tres que suben (con signo −), y se suma todo.

✏️ Ejemplo con Sarrus

A = (3 5 2; 4 2 3; −1 2 4)

|A| = (3)(2)(4)+(5)(3)(−1)+(2)(4)(2) − (−1)(2)(2)−(2)(3)(3)−(4)(4)(5)

|A| = 24−15+16+4−18−80 = −69

⚠️ Ojo

La Regla de Sarrus sólo funciona para 3×3. Para 4×4 o más grande da un resultado equivocado — ahí hay que usar cofactores.

3. Menor y cofactor

📌 Menor Mᵢⱼ

Dada una matriz A de n×n, el menor Mᵢⱼ es el determinante de la matriz (n−1)×(n−1) que queda al eliminar la fila i y la columna j de A.

📌 Cofactor Aᵢⱼ

El cofactor Aᵢⱼ es el menor con un signo pegado, según la posición: Aᵢⱼ = (−1)^(i+j) · Mᵢⱼ. El signo es + si i+j es par, y − si i+j es impar (como un tablero de ajedrez).

✏️ Ejemplo — menores de una 3×3

A = (2 −1 4; 0 1 5; 6 3 −4). Para M₁₃, tachá la fila 1 y la columna 3: queda |0 1; 6 3| . Para M₃₂, tachá la fila 3 y la columna 2: queda |2 4; 0 5|.

i = 1
j = 3

Elegí i y j: se tachan esa fila y columna, y queda marcado el menor Mᵢⱼ resultante (con su signo de cofactor).

4. Determinante n×n

La idea de "expandir por la primera fila" se generaliza a cualquier tamaño: el determinante es la suma de cada elemento de una fila (o columna) multiplicado por su cofactor.

Expansión por cofactores det A = a₁₁A₁₁ + a₁₂A₁₂ + ⋯ + a₁ₙA₁ₙ = Σₖ a₁ₖA₁ₖ
📌 Para recordar

Se puede expandir por cualquier fila o columna, no sólo la primera — el resultado es siempre el mismo. Conviene elegir la fila o columna con más ceros, para ahorrar cuentas: cada cero anula un término entero de la suma.

Calcular a mano crece rápido: un 4×4 pide cuatro determinantes de 3×3; un 5×5 pide cinco de 4×4 (=veinte de 3×3). Por suerte, las matrices triangulares dan un atajo enorme.

5. Determinante de una matriz triangular

📌 Triangular superior / inferior / diagonal

Triangular superior: todo lo que está debajo de la diagonal es cero. Triangular inferior: todo lo que está arriba es cero. Diagonal: cero en todos lados salvo la diagonal (es ambas a la vez).

Teorema — determinante triangular Si A es triangular (superior o inferior): det A = a₁₁ · a₂₂ · a₃₃ ··· aₙₙ

El determinante es, simplemente, el producto de la diagonal. Se demuestra expandiendo por cofactores repetidamente: en cada paso, todos los términos salvo el de la diagonal se anulan porque multiplican por un cero.

Tocá un caso: se resalta la diagonal (el único producto que sobrevive) y se calcula det A.

📌 Consecuencia directa

Una matriz triangular es invertible si y sólo si ningún elemento de su diagonal es cero (porque el producto de la diagonal, que es el determinante, sería 0 apenas uno de ellos lo sea).

6. Interpretación geométrica (2×2)

Si A es 2×2 con columnas (o filas) a=(a;c) y b=(b;d), esos dos vectores forman un paralelogramo con un vértice en el origen.

Teorema — área generada por A Área del paralelogramo = |det A|

El valor absoluto del determinante es exactamente esa área — sin necesidad de fórmulas de trigonometría.

a = 3,0
c = 1,0
b = 1,0
d = 4,0  |  det A = 11,00  |  Área = 11,00

Movés los cuatro deslizadores y el paralelogramo (sombreado) se actualiza junto con |det A| — siempre coinciden.

🗂️ Resumen del módulo
  • 2×2: det A = a₁₁a₂₂−a₂₁a₁₂.
  • 3×3: expansión por cofactores, o el atajo de Sarrus (sólo para 3×3).
  • Menor Mᵢⱼ: se tacha fila i y columna j. Cofactor Aᵢⱼ = (−1)^(i+j)Mᵢⱼ.
  • n×n: expansión por cofactores por cualquier fila o columna (conviene la de más ceros).
  • Triangular: det = producto de la diagonal. Invertible ⇔ det ≠ 0 ⇔ ninguna entrada diagonal es 0.
  • 2×2 geométrico: |det A| = área del paralelogramo generado por sus vectores fila/columna.
Por qué importa El criterio "det ≠ 0 ⇔ invertible" es la puerta de entrada a resolver sistemas de ecuaciones lineales con solución única — la vas a volver a usar todo el cuatrimestre.
Ojo con esto |A+B| ≠ |A|+|B| en general: el determinante no es lineal respecto a la suma de matrices completas (sí lo es fila por fila, pero no toda la matriz de una vez).
