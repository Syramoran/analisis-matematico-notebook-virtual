Definición formal (ε – δ)

"Se acerca tanto como queramos" suena claro, pero es informal. Acá le ponemos números precisos a esa idea.

La definición formal

📖 Definición (ε – δ)

Sea f definida en un intervalo abierto que contiene a a, excepto posiblemente en a. Decimos que

limx→a f(x) = L

si para todo número ε > 0 existe un número δ > 0 tal que

si  0 < |x − a| < δ   entonces   |f(x) − L| < ε

En criollo: vos elegís qué tan cerca de L querés estar (ε), y la definición te garantiza que existe una cercanía a a (δ) que lo logra — para cualquier ε que hayas elegido, no solo para uno.

Recorrámosla paso a paso

Los mismos elementos —franja de ε, franja de δ, el punto (a, L), la curva f(x)— aparecen siempre. Elegí un caso y andá avanzando para ver cómo se arman, y qué pasa cuando la definición no se cumple.

x = 2,50
Paso 1

Motivación: el fabricante de piezas cuadradas

Un fabricante pide piezas cuadradas de 100 cm² de área, con una tolerancia de ± 4 cm². ¿Qué largos de lado x son aceptables?

ε = 4 cm²

Razonamiento con ε = 4

Queremos |f(x) − L| < ε, es decir |x² − 100| < 4, o sea 96 < x² < 104. Tomando raíz cuadrada: 9,798 < x < 10,198.

Como el lado ideal es x=10, la distancia admisible hacia la izquierda es 10 − 9,798 = 0,202 y hacia la derecha 10,198 − 10 = 0,198. Nos quedamos con la más chica para que ambos lados funcionen: δ = 0,198.

Por qué el mínimo δ tiene que funcionar para los dos lados a la vez. Si uno admite más margen que el otro, hay que quedarse con el más exigente (el menor).
La regla del juego Si tomo un ε más chico, voy a necesitar un δ más chico.
Esa relación — "para cada ε existe un δ" — es exactamente la definición formal.

Volviendo al gráfico general: jugá con los controles para confirmar la relación entre ε y δ.

ε = 1,4 δ = 1,0

Franja horizontal = (L−ε, L+ε). Franja vertical = (a−δ, a+δ). Si la curva se escapa de la franja horizontal dentro de la franja vertical, ese δ no sirve para ese ε.

Ejemplo con álgebra exacta

Probemos que limx→3 (4x − 5) = 7 encontrando δ en función de ε:

Desarrollo

Queremos: si 0 < |x−3| < δ entonces |(4x−5) − 7| < ε.

|4x − 12| < ε  ⟹  4|x − 3| < ε  ⟹  |x − 3| < ε/4

Esto sugiere elegir δ = ε/4. Como la función es lineal, ¡este δ funciona exactamente, sin aproximar!

ε = 2,0  →  δ = ε/4 = 0,5

Otras versiones formales (mismo espíritu, distinto contexto)

Lateral izquierdo limx→a⁻f(x)=L: para todo ε>0 existe δ>0 tal que si a−δ<x<a entonces |f(x)−L|<ε.
Lateral derecho limx→a⁺f(x)=L: para todo ε>0 existe δ>0 tal que si a<x<a+δ entonces |f(x)−L|<ε.
Límite infinito limx→af(x)=∞: para todo M>0 existe δ>0 tal que si 0<|x−a|<δ entonces f(x)>M.
Límite al infinito limx→∞f(x)=L: para todo ε>0 existe N tal que si x>N entonces |f(x)−L|<ε.

Vas a usar la última (con N) en el próximo módulo, cuando veamos límites al infinito.

🖊️ Para pensar

En el ejemplo del fabricante, ¿por qué tuvimos que tomar el menor entre 0,202 y 0,198 y no el promedio o el mayor?

Ver una posible respuesta

Porque δ tiene que garantizar la tolerancia en ambas direcciones simultáneamente. Si eligiéramos 0,202 (el mayor), del lado derecho el margen se pasaría de lo permitido. El δ tiene que ser el más chico de los dos para que ningún lado se exceda.
