Demostrar ε–δ cuando la función no es lineal

Con 4x−5 encontrar δ = ε/4 fue directo porque la función es lineal. Con funciones cuadráticas o con raíces, hay un truco extra: acotar un factor molesto con una constante. Veamos los dos ejemplos que trae Stewart.

Ejemplo 1 — demostrar que lim(x→0) √x = 0

Paso 1: intuir δ

Queremos: si 0 < x < δ entonces |√x − 0| < ε, es decir √x < ε. Elevando al cuadrado ambos lados (válido porque ambos son positivos): x < ε².

Esto sugiere elegir δ = ε².

Paso 2: verificar que funciona

Dado ε>0, sea δ=ε². Si 0 < x < δ = ε², entonces √x < √(ε²) = ε, es decir |√x − 0| < ε. ✔ Funciona para cualquier ε>0, así que queda demostrado.

Fuente: Stewart, Sección 2.4, Ejemplo 3 (pág. 113-114).

ε = 0,60  →  δ = ε² = 0,36

Con δ=ε², la franja vertical (0, δ) queda exactamente contenida dentro de lo que hace falta para que √x cumpla la franja horizontal (0, ε). Probá cambiar ε y mirá cómo δ se ajusta como su cuadrado — no como una simple fracción, porque la curva no es una recta.

Ejemplo 2 — demostrar que lim(x→3) x² = 9

Acá aparece el truco central de las demostraciones con funciones no lineales: acotar un factor con una constante C.

Paso 1: relacionar |x²−9| con |x−3|

|x² − 9| = |x−3|·|x+3|. Si pudiéramos acotar |x+3| ≤ C para algún número C, entonces |x²−9| ≤ C|x−3|, y bastaría pedir |x−3| < ε/C.

¿Cómo acotamos |x+3|? Restringimos la atención a x cercanos a 3 — digamos, a menos de 1 unidad: |x−3| < 1. Eso implica 2 < x < 4, y por lo tanto 5 < x+3 < 7, así que |x+3| < 7 = C.

Ahora hay dos condiciones sobre δ: que garantice |x−3|<1 y que garantice |x−3| < ε/7. Nos quedamos con la más exigente (la menor): δ = mín{1, ε/7}.

Paso 2: verificar

Si 0<|x−3|<δ, entonces |x−3|<1 (así que |x+3|<7) y |x−3|<ε/7. Multiplicando: |x²−9| = |x−3||x+3| < (ε/7)·7 = ε. ✔

Fuente: Stewart, Sección 2.4, Ejemplo 4 (pág. 114-115).

ε = 2,50

δ = mín{1, ε/7} se recalcula solo con cada ε. Fijate que para ε chico, ε/7 es el que manda (δ=ε/7); pero si ε fuera muy grande, δ quedaría "topeado" en 1 — el candado de seguridad que impusimos para poder acotar |x+3|.

La receta general Para demostrar lim(x→a) f(x) = L con f no lineal: (1) escribí |f(x)−L| en términos de |x−a| multiplicado por algún factor, (2) restringí x a un entorno de a (por ejemplo |x−a|<1) para acotar ese factor con una constante C, (3) tomá δ = mín{1, ε/C}.
🖊️ Para pensar

¿Por qué elegimos restringir con "|x−3|<1" y no, por ejemplo, "|x−3|<0,5" o "|x−3|<5"? ¿Cambiaría el resultado final si usáramos otro número?

Ver una posible respuesta

El número "1" es arbitrario — cualquier cota funcionaría, solo cambiaría el valor de C y por lo tanto la fórmula final de δ. Por ejemplo, con |x−3|<0,5 tendríamos 2,5<x<3,5, entonces 5,5<x+3<6,5, y C=6,5 en vez de 7. El resultado (que el límite es 9) no cambia — solo cambia la "receta" concreta de δ que demuestra ese resultado. Se elige 1 porque da números enteros cómodos, no porque sea matemáticamente especial.
